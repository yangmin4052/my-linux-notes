| 快捷键 | 备注                     |
| ------ | ------------------------ |
| ctrl+c | 强制终止当前命令         |
| ctrl+l | 清屏                     |
| ctrl+a | 光标移动到命令行首       |
| ctrl+e | 光标移动到命令行尾       |
| ctrl+u | 从光标所在位置删除到行首 |
| ctrl+k | 从光标所在位置删除到行尾 |
| ctrl+z | 把命令放入后台           |
| ctrl+r | 在历史命令中搜索         |
